Administration des systèmes et des réseaux (Option SISR)

Réalisation 01 : Haute Disponibilité & Services Systèmes

Organisation fictive : Resilience-IT

Contexte : Mise en place d'une infrastructure virtualisée en cluster pour garantir la continuité de service et centraliser les identités[cite: 191].


Configurations réalisées :

  • Cluster de virtualisation (Proxmox/ESXi) en Haute Disponibilité (HA).
  • Stockage centralisé iSCSI via TrueNAS.
  • Contrôleur de domaine Active Directory (Windows Server 2022).
  • Gestion des Unités d'Organisation (OU) et stratégies GPO.
  • Service d'inventaire GLPI lié à l'annuaire via le protocole LDAP.

Proxmox VE Windows Server 2022 TrueNAS Active Directory

Réalisation 02 : Sécurisation Réseau & Accès Nomades

Organisation fictive : Nomad-Security

Contexte : Déploiement d'un accès distant sécurisé chiffré lié à l'annuaire centralisé de la réalisation n°01.


Configurations réalisées :

  • Passerelle VPN sécurisée via le protocole WireGuard.
  • Micro-virtualisation par conteneurs avec Docker & Docker Compose.
  • Authentification Multi-Facteurs (MFA) avec Authelia lié à l'Active Directory.
  • Reverse Proxy Nginx pour l'accès sécurisé aux applications web internes.
